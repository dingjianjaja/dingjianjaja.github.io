    字符串

    在 Swift 中绝大多数的情况下，推荐使用 String 类型

    使用 String 的原因

    • String 是一个结构体，性能更高
      • String 目前具有了绝大多数 NSString 的功能
      • String 支持直接遍历
    • NSString 是一个 OC 对象，性能略差
    • Swift 提供了 String 和 NSString 之间的无缝转换
